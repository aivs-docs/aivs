SpeechSynthesizer接口

本接口基于Alexa Voice Service v20160207进行定制， 兼容AVS SpeechSynthesizer Interface version1.0

Speak Directive

举例

消息体 Sample Message

{
    "directive": {
        "header": {
            "namespace": "SpeechSynthesizer",
            "name": "Speak",
            "messageId": "36efe705-3ce6-4285-9648-3c40dfcabac7",
            "dialogRequestId": "9df8b356-63bb-4741-a44b-d46bf159156b"
        },
        "payload": {
            "url": "cid:a0c669e1-3308-436a-8ff6-429c037753d5",
            "text": "TTS文本内容",
            "format": "AUDIO_MPEG",
            "remoteUrl":"http://url.mp3",
            "token": "a0c669e1-3308-436a-8ff6-429c037753d5"
        }
    }
}

二进制音频附件 Binary Audio Attachment

Content-Type: application/octet-stream
Content-ID: {{Audio Item CID}}

{{BINARY AUDIO ATTACHMENT}}

参数描述

参数名 说明 数据类型
messageId 用于表示特定消息的唯一ID string
dialogRequestId 与下发指令相关联的唯一ID string
url 音频内容的唯一标识符，url始终遵循前缀 例如：cid: string
remoteUrl 音频内容对应的mp3下载地址 string
text TTS文本内容 string
format 返回的音频格式，目前支持：AUDIO_MPEG string
token 表示TTS对象的标记token string

SpeechStarted Event

举例

消息体 Sample Message

{
  "event" : {
    "header" : {
      "namespace" : "SpeechSynthesizer",
      "name" : "SpeechStarted",
      "messageId" : "2aef730b-39f3-4590-8d8b-bb5c41e1c496"
    },
    "payload" : {
      "token" : "a0c669e1-3308-436a-8ff6-429c037753d5"
    }
  }
}

参数描述

参数名 说明 数据类型
messageId 用于表示特定消息的唯一ID string
token Speak Directive提供的标识符 string
