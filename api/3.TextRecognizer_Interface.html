TextRecognizer

TextRecognizer接口

本接口提供文本语义识别和理解的功能，并返回相对应的识别结果

音频识别 TextRecognize Event

举例

消息体 Sample Message

{
  "event" : {
    "header" : {
      "namespace" : "TextRecognizer",
      "name" : "TextRecognize",
      "messageId" : "72f5b7a1-1025-4299-95a1-8959f58deff9",
      "dialogRequestId" : "adda353e-d959-4f99-a6ec-8f66b4d8c1f4"
    },
    "payload" : {
      "language" : "zh-CN",
      "text" : "播放一组新闻"
    }
  }
}

参数描述

参数名 说明 数据类型
messageId 用于表示特定消息的唯一ID string
dialogRequestId 与下发指令相关联的唯一ID string
language 文本的语言类型，目前支持：zh-CN string
text 需要识别的文本内容 string

语音合成 TextToSpeech Event

举例

消息体 Sample Message

{
    "event": {
        "header": {
            "namespace": "SpeechRecognizer",
            "name": "TextToSpeech",
            "messageId": "dad1e0b9-a961-4f10-be90-996478584fa9",
            "dialogRequestId": "e67bcd6e-fd1c-4cbb-a405-c0b373843d77"
        },
        "payload": {
            "language": "zh-CN",
            "text": "播放三国演义"
        }
    }
}

参数描述

参数名 说明 数据类型
messageId 用于表示特定消息的唯一ID string
dialogRequestId 与下发指令相关联的唯一ID string
language TTS语言，目前支持：zh-CN string
text 需要播报的文本内容 string
