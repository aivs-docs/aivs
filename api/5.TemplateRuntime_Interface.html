TemplateRuntime

TemplateRuntime接口

本接口基于Alexa Voice Service v20160207进行定制， 兼容AVS TemplateRuntime Interface

RenderTemplate Directive

模版类型 Type

BodyTemplate1

消息体 Sample Message

{
  "directive" : {
    "header" : {
      "namespace" : "TemplateRuntime",
      "name" : "RenderTemplate",
      "messageId" : "a8f54567-e420-47d8-8b35-20ab9fbcfab1",
      "dialogRequestId" : "1bea9874-80d8-4372-997b-409f76fc8f70"
    },
    "payload" : {
      "token" : "0f889fe1-292d-48aa-b435-a0eea789da19",
      "type" : "BodyTemplate1",
      "title" : {
        "mainTitle" : "标题",
        "subTitle" : "副标题"
      },
      "textField" : "正文"
    }
  }
}

参数描述

参数名 说明 数据类型
messageId 用于表示特定消息的唯一ID string
dialogRequestId 与下发指令相关联的唯一ID string
token 表示当前模版的标识 string
type 模版类型，当前举例中为BodyTemplate1 string
title 包含对模版进行描述的多项Key/Value值 object
title.mainTitle 标题 string
title.subTitle 副标题 string
textField 正文 string

{
  "directive" : {

      "namespace" : "TemplateRuntime",
      "name" : "RenderTemplate",

      "title" : {
        "mainTitle" : "标题",
        "subTitle" : "副标题"
      },
      "textField" : "正文",
      "image" : {
        "sources" : [ {
          "url" : "http://image-url",
          "widthPixels" : 300,
          "heightPixels" : 300
        } ]
      }
    }
  }
}

参数描述

参数名 说明 数据类型

dialogRequestId 与下发指令相关联的唯一ID string
token 表示当前模版的标识 string
type 模版类型，当前举例中为BodyTemplate2 string
title 包含对模版进行描述的多项Key/Value值 object
title.mainTitle 标题 string
title.subTitle 副标题 string
textField 正文 string
image 包含对图片进行描述的多项Key/Value值 image structure

WeatherTemplate

消息体 Sample Message

{
  "directive" : {
    "header" : {
      "namespace" : "TemplateRuntime",
      "name" : "RenderTemplate",
      "messageId" : "47ca2d08-5cca-4513-916f-3788b09d724b",
      "dialogRequestId" : "2c871805-a46c-43c8-b8ea-4a108d18303e"
    },
    "payload" : {
      "token" : "c7804109-ad9d-48fe-a1bf-5d1581ac249e",
      "type" : "WeatherTemplate",
      "title" : {
        "mainTitle" : "天气",
        "subTitle" : "Nov21 Wed"
      },
      "currentWeather" : "7",
      "description" : "天气描述",
      "highTemperature" : {
        "value" : "9"
      },
      "lowTemperature" : {
        "value" : "-3"
      },
      "weatherForecast" : [ {
        "date" : "Nov 22",
        "highTemperature" : "8",
        "lowTemperature" : "-3"
      }, {
        "date" : "Nov 23",
        "highTemperature" : "5",
        "lowTemperature" : "-5"
      } ]
    }
  }
}

参数描述

参数名 说明 数据类型
messageId 用于表示特定消息的唯一ID string
dialogRequestId 与下发指令相关联的唯一ID string
token 表示当前模版的标识 string
type 模版类型，当前举例中为WeatherTemplate string
title 包含对模版进行描述的多项Key/Value值 object
title.mainTitle 标题 string
title.subTitle 副标题 string
currentWeather 当前气温 string
description 当前天气描述 string
highTemperature 包含对模版进行描述的多项Key/Value值 object
hightemperature.value 当前最高气温 string
lowTemperature 包含对模版进行描述的多项Key/Value值 object
lowTemperature.value 当前最低气温 string
weatherForecast 未来N天的天气预报 list
weatherForecast.date 预报日期，格式为ABBR_MONTH_DAY，例如"Nov 22" string
weatherForecast.highTemperature 最高气温 string
weatherForecast.lowTemperature 最低气温 string

RenderPlayerInfo Directive

举例

消息体 Sample Message

{
  "directive" : {
    "header" : {
      "namespace" : "TemplateRuntime",
      "name" : "RenderPlayerInfo",
      "messageId" : "5049d7a5-44c1-4bf7-b17b-1107f2ba9ae6",
      "dialogRequestId" : "c8da5b2b-672f-452a-8e83-b5f4192c4f67"
    },
    "payload" : {
      "audioItemId" : "xiaoai:7773234311568433024",
      "content" : {
        "title" : "标题",
        "titleSubtext1" : "副标题",
        "header" : "三级标题",
        "mediaLengthInMilliseconds" : 1324414,
        "art" : {
          "sources" : [ {
            "url" : "http://image-url",
            "widthPixels" : 300,
            "heightPixels" : 300
          } ]
        },
        "provider" : {
          "name" : "xiaoai"
        }
      },
      "controls" : [ {
        "type" : "BUTTON",
        "name" : "NEXT",
        "enabled" : true,
        "selected" : false
      }, {
        "type" : "BUTTON",
        "name" : "PREVIOUS",
        "enabled" : true,
        "selected" : false
      }, {
        "type" : "BUTTON",
        "name" : "PLAY_PAUSE",
        "enabled" : true,
        "selected" : false
      } ]
    }
  }
}

参数描述

参数名 说明 数据类型
messageId 用于表示特定消息的唯一ID string
dialogRequestId 与下发指令相关联的唯一ID string
audioItemId 表示当前模版的标识 string
content 包含对模版进行描述的多项Key/Value值 object
content.title 标题 string
content.titleSubtext1 副标题 string
content.header 三级标题 string
content.mediaLengthInMilliseconds 音频流的长度 string
content.art 该模版包含的图片 image structure
content.provider 内容提供商 object
content.provider.name 内容提供商名称 string
controls 屏幕上显示的控制按钮 list
controls.type 按钮类型，包括：BUTTON，TOGGLE string
controls.name 按钮名称，包括：PLAY_PAUSE, NEXT, PREVIOUS, SKIP_FORWARD, SKIP_BACKWARD, SHUFFLE, LOOP, THUMBS_UP, THUMBS_DOWN string
controls.enabled 按钮是否可以点击 boolean
controls.selected 按钮是否被选中 boolean

Image Structure

举例

格式

    {
        "sources" : [ {
          "url" : "http://image-url",
          "widthPixels" : 300,
          "heightPixels" : 300
        } ]
    }

参数描述

参数名 说明 数据类型
sources 图片列表 list
sources[i].url 与下发指令相关联的唯一ID string
sources[i].widthPixels 图片宽度，单位是像素。该参数由内容提供商决定，可能会为空 string
sources[i].heightPixels 图片高度，单位是像素。该参数由内容提供商决定，可能会为空 object
